The design language across all Ken's projects is sci-fi command deck:

- **Colors:** Deep black (#08090d), electric cyan (#00e5ff / #00f0ff), purple (#7c4dff), neon green (#00e676 / #00ff88), red (#ff1744 / #ff4466), amber (#ffd740 / #f0a030)
- **Feel:** "Alien Isolation meets Bloomberg Terminal meets JARVIS"
- **Effects:** Scan lines, glow on hover, grid overlays, typing animations
- **Mascot:** Robot face "Pipe-R" with expressions and eye tracking

**Why:** Consistent across hub.js terminal palette, ForgeAgent TUI, and web UI spec. This is Ken's aesthetic — not negotiable.

**How to apply:** Any new UI element should use these colors and effects. Don't introduce flat/material/generic design.
