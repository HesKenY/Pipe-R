# Android Copy Source

The canonical Android project for REVV Main lives at:

- `android/`

Use this folder as the source of truth when you need to copy the Android app out of the master repo.

## Copy Destination Used By The Website Tree

- `site/apps/revv-main/android/`

## Helper Script

Use:

```powershell
.\scripts\copy-android-project.ps1
```

That copies the Android project into the internal REVV app folder so the backup layout stays organized.
